# 🚗 Детектор свободных парковочных мест (Умная парковка)

Проект машинного обучения для бинарной классификации парковочных мест (Свободно / Занято) по фрагментам изображений с камер видеонаблюдения.

## 📊 Результаты обучения моделей

В ходе работы было проведено исследование и сравнительный анализ 5 различных архитектур сверточных нейросетей. Обучение проводилось на адаптированной выборке из датасета PKLot (20 000 изображений, 2 класса: `Empty`, `Occupied`).

| Модель | Точность (Accuracy) | Функция потерь (val_loss) | Примечание |
| :--- | :--- | :--- | :--- |
| **EfficientNetB0** | **99.90%** 🏆 | **0.0031** | Абсолютный лидер, используется в production |
| MobileNetV2 | 99.65% | 0.0087 | Лучший баланс скорости и точности |
| ResNet18 | 99.60% | 0.0137 | Классический Transfer Learning |
| CustomCNN | 99.30% | 0.0197 | Собственная архитектура (обучена с нуля) |
| SqueezeNet | 98.80% | 0.0477 | Базовый baseline |

## 📁 Структура проекта и скрипты

В репозитории сохранен весь пайплайн разработки: от подготовки сырых данных до развертывания веб-интерфейса.

* `app.py` — главный файл веб-приложения (Streamlit). Выполняет инференс (предсказание) с использованием лучшей обученной модели.
* `train.py` — основной скрипт обучения всех архитектур нейросетей с логированием метрик.
* `prepare_data.py` — скрипт предварительной обработки: разделение исходного датасета на тренировочную (16 000) и валидационную (2 000) выборки.
* `gather_images.py` — утилита для автоматизированного сбора/перемещения нужных классов изображений из сырого датасета.
* `cache_models.py` — скрипт для предварительной загрузки весов предобученных моделей (Transfer Learning).
* `test_gpu.py` — диагностический скрипт для проверки доступности CUDA/CPU в PyTorch.
* `models/` — директория для сохранения весов обученных моделей (`.pth`).

## 🛠 Стек технологий

* **Machine Learning:** PyTorch, Torchvision
* **Frontend / UI:** Streamlit
* **Data Processing:** Pillow (PIL), NumPy
* **Окружение:** Python 3.10+

## 🚀 Установка и запуск

1. Клонируйте репозиторий на свой локальный компьютер:

   ```bash
   git clone <ссылка_на_ваш_репозиторий>
    ```

2. Перейдите в директорию проекта и активируйте виртуальное окружение:

    ```bash
    cd Praktic
    python -m venv venv
    # Для Windows:
    venv\Scripts\activate
    # Для Linux/macOS:
    source venv/bin/activate
    ```

3. Установите необходимые зависимости:

    ```bash
    pip install -r requirements.txt
    ```

4. Запустите веб-приложение:

    ```bash
    streamlit run app.py
    ```

После запуска в вашем браузере автоматически откроется интерфейс приложения по адресу `http://localhost:8501`.
